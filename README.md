## 🎯 프로젝트 소개

**관광 실무 전문가 18명의 설문 결과를 100% 반영한 실전형 AI 컨시어지**

실제 네이버 리뷰 데이터를 분석하여 재방문율, 평균 방문 횟수, 긍정 평가율을 기반으로 신뢰할 수 있는 추천을 제공합니다.

> 📍 **현재 제공 지역**: 춘천  
> 🚀 **확대 예정**: 강릉, 속초, 평창, 원주 등 강원도 전역

### 🌟 핵심 차별점

- ✅ **실제 리뷰 데이터**: 수백~수천 개의 네이버 리뷰 분석
- ✅ **똑똑한 일정 생성**: 매번 다른 조합, 중복 없는 다양한 일정
- ✅ **설문 기반 기능**: 관광 전문가 18명의 실무 니즈 반영
- ✅ **투명한 통계**: 평균 방문 횟수, 긍정 평가율 등 신뢰도 지표

---

## 🚀 5가지 핵심 기능

### 💬 1. AI 관광 챗봇
**실제 리뷰 기반 답변**

**특징:**
- 재방문율 우선 추천
- 통계 데이터 포함
- 실제 리뷰 내용 요약

---

### 📋 2. 자동 일정 생성기
**매번 새로운 조합의 똑똑한 일정**

**특징:**
- 🎲 매번 다른 일정 (중복 방지)
- ⭐ 상위권 장소만 선택 (품질 보장)
- 🕐 시간대별 최적화
- 📥 텍스트 다운로드

**알고리즘:**
1. 상위 30개 장소 풀 생성
2. 중복 없이 랜덤 선택
3. 시간대별 최적 배치
4. 다양성과 품질 동시 확보

---

### 🏆 3. TOP 추천
**3가지 기준으로 정렬 가능**

**정렬 기준:**
- 📈 재방문율: 2번 이상 방문한 비율
- 👍 긍정 평가: 긍정 키워드 포함 리뷰 비율
- 📊 리뷰 수: 방문자가 많은 곳

**특징:**
- 최근 리뷰 미리보기
- 카테고리별 필터링
- 실시간 통계 표시

---

### 📊 4. 장소 비교 분석
**2개 장소 상세 비교**

**4가지 지표 비교:**
1. 총 리뷰 수
2. 재방문율
3. 평균 방문 횟수
4. 긍정 평가율

**특징:**
- 객관적 데이터 기반 비교
- 승자 자동 판정
- 의사결정 지원

---

### ⭐ 5. 리뷰 통계 대시보드
**전체 데이터 통계 분석**

**특징:**
- 실시간 통계 집계
- 카테고리별 트렌드
- 신뢰도 높은 정보

## 🛠️ 기술 스택

### Core
- **Frontend**: Streamlit 1.30+
- **AI Model**: OpenAI GPT-4o-mini / GPT-5-nano
- **Framework**: LangChain 0.1+
- **Vector DB**: ChromaDB
- **Embeddings**: OpenAI Embeddings

### Data
- **Source**: 네이버 지도 리뷰 (xlsx)
- **Categories**: 맛집, 명소, 카페, 병원
- **Volume**: 10,000+ 리뷰

### Optimization
- **Token**: 74% 감소 (728K → 180K)
- **Caching**: Streamlit cache_data/cache_resource
- **Batch**: 최적화된 배치 처리

## 📊 데이터 정보

### 네이버 리뷰 데이터

> 📍 **현재 제공 지역: 춘천**

| 카테고리 | 장소 수 | 리뷰 수 | 파일 형식 |
|----------|---------|---------|-----------|
| 맛집 | 30+ | 4,000+ | xlsx |
| 명소 | 25+ | 5,000+ | xlsx |
| 카페 | 25+ | 2,500+ | xlsx |
| 병원 | 5+ | 500+ | xlsx |
| **합계** | **85+** | **12,000+** | |

### 리뷰 데이터 구조

```
Excel 파일 (naver_review_[장소명].xlsx)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
컬럼:
- store: 장소명
- date: 작성일
- nickname: 작성자
- content: 리뷰 내용
- revisit: 재방문 정보 ("2번째 방문", "3번째" 등)
```

### 통계 지표

- **재방문율**: 2번째 이상 방문한 리뷰 비율
- **평균 방문**: 리뷰어들의 평균 방문 횟수
- **긍정 평가**: 긍정 키워드 포함 리뷰 비율

## 💡 사용 시나리오

### 시나리오 1: 가족 여행 계획

```
1. [일정 생성기] 탭 선택
2. 여행 기간: "2박 3일"
3. 카테고리: 맛집 + 명소 + 카페
4. 우선순위: "재방문율"
5. "일정 생성" 버튼 클릭
6. 생성된 일정 확인
7. 텍스트 다운로드
8. 다시 생성하면 다른 조합!
```

### 시나리오 2: 맛집 찾기

```
1. [TOP 추천] 탭 선택
2. 카테고리: "맛집 리뷰"
3. 정렬 기준: "재방문율"
4. 상위 20개 확인
5. 통계 비교
6. 최근 리뷰 확인
7. 최종 선택
```

### 시나리오 3: 두 곳 고민

```
1. [장소 비교] 탭 선택
2. 장소 1: "1.5닭갈비"
3. 장소 2: "798삼천동고깃집"
4. "비교하기" 클릭
5. 4개 지표 확인
6. 승자 참고하여 결정
```

### 시나리오 4: AI에게 질문

```
[AI 챗봇]에서:
"재방문율 80% 이상인 강릉 명소 추천해줘"

→ 통계와 함께 상세 답변
→ 실제 리뷰 내용 포함
→ 평균 방문 횟수 표시
```
